# Stochatic Simulation Projects

This repository contains three computational modeling projects completed as coursework of Stochastic Simulation.

## Projects

### Project 1: Monte Carlo Geometry
This project estimates the intersection volume of a sphere and a torus using Monte Carlo integration. It also compares uniform random sampling with a deterministic logistic-map sequence and studies an off-center torus using a mixed sampling idea.

### Project 2: Airport Queue Simulation
This project models a representative airport security lane with discrete-event simulation. It validates the simulator against queueing theory and compares operational interventions such as adding servers or reducing service-time variability.

### Project 3: Option Pricing and Delta Hedging
This project studies European call pricing and hedging under the Black–Scholes framework and several extensions. It includes Monte Carlo pricing, antithetic variance reduction, delta hedging with execution delays, and model-risk analysis under jump and regime-switching dynamics.
